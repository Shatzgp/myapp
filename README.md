# Dating Site Project

Welcome to the Dating Site project! This project is a web-based dating application built using Python, Flask, JavaScript, and CSS.

## Project Overview

The Dating Site project aims to provide a platform for users to connect with each other, find potential matches, and build meaningful relationships. The project includes the following features:

- User registration: Users can create an account by providing their name, age, gender, location, and bio.
- Profile customization: Users can upload photos and personalize their profiles.
- Matching algorithm: The application employs an algorithm to match users based on their preferences and interests.
- User interaction: Users can browse profiles, send messages, and engage in conversations.

## Project Structure

The project repository is organized as follows:

- `server.py`: The main Python file containing the Flask application and server logic.
- `templates/`: This directory contains HTML templates used for rendering the web pages.
- `static/`: This directory contains static assets such as CSS stylesheets and JavaScript files.

## Setup and Installation

To set up the project on your local machine, follow these steps:

1. Clone the repository:
   ```shell
   git clone https://github.com/your-username/dating-site-project.git
   ```

2. Install the dependencies:
   ```shell
   pip install -r requirements.txt
   ```

3. Start the Flask server:
   ```shell
   python server.py
   ```

4. Open your web browser and navigate to `http://localhost:5000` to access the dating site.
